Arizona Pig Rescue

Full-CRUD Web Application
Express.js, Node.js, MySQL, Sequelize, Handlebars, Bootstrap, HTML5, CSS3

Arizona Pig Rescue is a re-design of the Ironwood Pig Sanctuary website. After learning about the "mini-pig" myth and the problems it has created, my team did some research and found the amazing folks at the sanctuary, and we decided to rebuild their website.

I added database functionality using MySQL and Sequelize, implementing an Administrator Dashboard that allows the sanctuary owners to keep track of the pigs they have on their roster. They can add new pigs, update the adoption status of any pig they have, or remove individual pigs from their database. The dashboard also allows them to track individuals who have expressed interest in either volunteering or adpoting a pig from their farm.

I used the Handlebars.js templating engine to render the results from the Sequelize queries to the front-end. The front-end was styled by my teammates using Bootstrap.

I'm still working on adding functionality to the website, and we plan on donating it to the Ironwood Pig Sanctuary in hopes of supporting the wonderful work they do! Please check out the website demo(administrator password: 'pigministrator').

Out And About

Web Application
Node.js, Google Maps API, Google Places API, OpenWeather API, Bootstrap, HTML5, CSS3

Out and About is a web application that provides the user a map of local bars in a zip code they search. It also provides a snapshot of current weather conditions in that area, using the OpenWeather API.

It uses the Google Maps and Google Places APIs to populate info-marker pins on each establishment. These info-markers include the establishment's name, address, whether or not it's currently open, and rating(out of 5) by other Google Users.
